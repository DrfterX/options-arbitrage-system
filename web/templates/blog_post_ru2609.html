二、先回顾：N 型结构是什么？

一个 N 型结构由 A、B、C 三个点构成：

上升 N 型（做多信号）：

  • A 点 = 第一笔起点，是整个 N 型的最低点
  • A → B = 第一笔上涨，B 是这段区间的最高点
  • B → C = 第二笔回调，C 是回调的低点
  • 关键规则：C 点不能低于 A 点，否则上升 N 型被破坏
  • C → 最新价 = 正在形成的第三笔（N 型向上破位方向）

下降 N 型（做空信号）同理反向。

核心逻辑：先判断方向，再按方向标点。不是简单地找"前高前低"，而是按第一笔→第二笔→第三笔的逻辑递进。这也是信号矩阵 N 型引擎的识别算法基础——状态机逐笔扫描，而不是依赖形态模板匹配。

五、这个数据是怎么来的？

上面所有的 N 型结构数据——A/B/C 三点的精确定位、多周期信号方向判断、结构状态实时跟踪——都不是手工画的，全部由信号矩阵的 N 型结构自动识别引擎 实时生成。

这个引擎的核心是一条 400 行不到的状态机流水线：

  1. 从 K 线数据中提取局部极值点（高点和低点）
  2. 用前向非重叠扫描算法识别出 A→B→C 三笔结构
  3. 按上升/下降方向验证 C 点是否破 A
  4. 跨周期同步：对同一品种在 15 分钟到周线共 5 个周期上同时跑

这意味着：行情更新后，结构状态自动重算——不需要任何人工干预，每次打开页面看到的都是最新结构。
