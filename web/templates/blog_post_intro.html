一、为什么是"信号矩阵"？

核心思路很简单：

一个品种，N 个周期，每个周期判断多/空/震荡 → 看哪些周期方向一致 → 共振就是信号。

比如沪铜当前的状态：

15分偏多、30分偏多、60分钟震荡、日线偏多、周线偏多。

5 个周期里有 4 个指向多，这就是跨周期共振。

共振不是预测，是概率信号。周期越多指向同一方向，这个方向的概率优势就越大。

二、我们加入了 N 型结构自动识别

除了基本的信号矩阵，还做了 N 型结构自动识别。

啥是 N 型结构？期货价格走势大部分时间不是直线，而是 ABC 三段：

  • 上升 N 型：A 点最低 → 涨到 B 点 → 回调到 C 点（不破 A）→ 准备向上破位
  • 下降 N 型：A 点最高 → 跌到 B 点 → 反弹到 C 点（不过 A）→ 准备向下破位

手工画这些线很烦，但算法做这件事一秒都不要。

K 线上自动标注 ABC 点和线段，一眼看出当前结构状态。目前覆盖 15 个品种 × 5 个周期（15分/30分/60分/日线/周线），打开页面即可查看。

三、期权数据也放上去了

商品期权这几年越来越火，但散户能用的 IV 分析工具几乎为零。

我们做了 IV 百分位柱状图，看当前波动率在历史中的位置：

  • 🟢 低 IV（< 20%）：期权便宜 → 适合买入策略
  • 🟡 中 IV（20% – 80%）：价格适中 → 方向性价差
  • 🔴 高 IV（> 80%）：期权溢价高 → 适合卖出策略

还有策略评分系统，对常见期权策略打分——不构成交易建议，就是帮你过滤掉明显不划算的策略。

三个看板，一个账号通用： 信号主看板（多周期信号矩阵 + 共振检测）— 期货 K 线（N 型结构标注 + K 线图）— 期权分析（IV 百分位 + 策略评分）
