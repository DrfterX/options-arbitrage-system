## 一、最终确认的差距清单（5 项）

以下差距经用户逐条确认，认定为真实偏离项：

### 🔴 GAP #2：周期配置缺 3 分钟 MACD 验证

**原文要求：** 3分钟MACD蓝→红→蓝颜色变化验证15分钟N型结构的稳定性。不满足视作无效突破。

**当前现状：** `settings.py` 中 `LEVEL3_STABILITY_TIMEFRAME = "15m"`，检查的是15分钟MACD稳定性，不是3分钟。

**修正：** 将 Level3 MACD 稳定性检查从 15分钟 改为 **3分钟**，检查蓝→红→蓝完整颜色序列。


### 🔴 GAP #3：评分系统机制走样（核心偏离）

**原文要求：**
- 3分制的必要条件：大周期稳定（周线N型+日线MACD）+ 中周期稳定（1小时N型+15分钟MACD）+ 小周期稳定（15分钟N型+3分钟MACD）
- **每层各记1分，3分是硬性入场条件——缺一分都不行**
- 其他周期（如1小时MACD配合日线）出现 → 可以加分，但**不能用来补齐基础3分中的缺分**
- 3分入场，4分加仓

**当前现状：** 0~1.0浮点累加，中间有降级路径（WATCH/CANDIDATE），不满3分也能绕过。

**修正：**
- 改为离散3分制：大/中/小各1分，3分全满足才允许入场
- 加分体系单独计算（不影响入场条件）
- 删除所有降级路径，评分不足3分直接返回 NONE


### 🔴 GAP #4：MACD 条件被人为降级

**原文要求：** "没有MACD的红蓝变化配合的K线结构，是不稳定的结构……即使出现结构突破，也视为无效信号。"

**当前现状：** `scorer.py` 第 422 行注释：**"只要方向匹配即通过Level2，MACD轨迹变为评分分级因素而非硬门槛"**

**修正：** 删除放宽逻辑，MACD 蓝→红→蓝 轨迹验证恢复为阻断性条件。


### 🔴 GAP #5：止盈脱离结构推算

**原文要求：** 以大周期结构（周线）第一笔的最低点到最高点的幅度为标准，第三笔的最低点加上这个幅度作为止盈目标。天然获得 **1:10~1:30+** 的超大盈亏比。

**当前现状：** 固定 2:1 风险收益比（`tp_ratio = 2.0`）。

**修正：** 改为大周期第一笔幅度推算第三笔目标位。


### 🔴 GAP #7：评分重置完全缺失

**原文要求：**
- 上升趋势中，第4笔运行破坏了第2笔的终点 → 多单评分重置为0
- 上升过程中出现反向倒N型且达到开仓分数 → 多单评分重置为0

**当前现状：** 无评分重置逻辑，评分只会累加不会归零。

**修正：** 新增评分重置检查函数，在大周期结构破坏或反向结构达到开仓条件时重置对应方向的评分。
