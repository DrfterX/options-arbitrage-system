## 失败根因分析

### 1. Pullback 错过了优质信号

直接入场有 55 个信号，Pullback 只剩 30 个。丢失的 25 个信号来自：
- **强势突破无回撤**：趋势强劲时价格直接突破 B 点后继续前进，不回头测试
- 这些无回撤的突破恰恰是质量更高的信号

### 2. 进入价格更差

Pullback 要求二次突破入场，这意味着：
- 入场价格比首次突破更高（多头）/更低（空头）
- 止损 ATR 不变 → 风险-回报比恶化
- 胜率从 40% 降到 26.7%

仅 TA 品种 Pullback 接近直接入场水平。其他全部品种 Pullback 严重劣化。

## 决策建议

1. **放弃 P2 Pullback 入场方案** — 已证明在所有参数配置下均劣于直接入场
2. **保留直接入场（B 点首次突破）作为默认模式**
3. **当前最佳状态回退至 P1-B**：直接入场 + 混合止损（50.3 WATCH）
4. 下一阶段方向建议：参考 CEO/CTO 会议决定下一步策略方向
