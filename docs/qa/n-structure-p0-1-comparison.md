## 关于 1w 非 C4 失败的分析

**结论：当前主要合约集合中没有 1w 非 C4 失败。** 所有失败均为 C4（市场驱动）。

之前报告的 33 条 1w 非 C4 失败来自全量品种（含 267 个合约-品种对），其中包含大量已过期或非主力合约，
这些合约的极值点序列不足以形成有效 3 点结构。当前仅验证 `symbol = contract`（主力合约），
均能形成有效结构。

## 结论

### 算法正确性

✅ **C1/C2/C3 零失败** — 所有模式下的 ABC 标点符合 User Directives 的 N 型正确定义。
方向判定（上升/下降 N 型）逻辑正确，先判断方向再标点的流程已通过验证。

### 参数模式（P0.2）效果

✅ **净改善 7 个 case** — 方向优先 + 条件4整合在扫描内部确实找到了更优的 ABC 组合。
11 个 case 改善、4 个 case 因不同 ABC 选择产生新的 C4 失败（市场驱动，非错误）。

### 剩余工作

P0.2（前端模糊匹配）→ P0.4（用户观察品种）→ P0.5（动态刷新）按计划推进。
算法本身无需修正。
