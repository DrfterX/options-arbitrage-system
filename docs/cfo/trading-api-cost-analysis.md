### 1.2 方案详细分析

#### 方案 A：CTP 直连（推荐 — 路径 A）

| 项目 | 详情 |
|------|------|
| **接入方式** | 期货公司开通CTP极速交易通道，直接通过 CTP API (Python 封装如 `ctp-python` / `vnpy.ctp`) 下单 |
| **开户费** | 0 元 |
| **最低资金** | 多数期货公司要求 50 万以上申请CTP通道；部分中小公司如 **中信建投、海通期货** 可谈至 20 万 |
| **数据源** | CTP 行情(Level 1) + AKShare 辅助（当前已有） |
| **手续费** | 交易所手续费 + 期货公司加收（+0.01 至 +0.1元/手不等）。CTP通道客户通常可谈到 +0.01 |
| **月费** | 0 元（CTP 通道本身免费，极少数高频专用线路收费） |
| **优势** | 行业标准，延迟最低（毫秒级），全球最成熟的期货交易API之一 |
| **劣势** | 开户门槛高，API 文档为C++风格（Python封装不够完善），需自行实现风控 |
| **推荐度** | ⭐⭐⭐⭐⭐ — 长期实盘的最终归宿 |

| **优势** | 门槛极低，Python SDK 成熟，包含行情+交易+风控，无需自研交易终端 |
| **劣势** | QMT 是本地终端（需要有一台 Windows/Mac 常开），策略部署在QMT内或外部通过API调用 |
| **推荐度** | ⭐⭐⭐⭐⭐ — 零成本开始实盘的最佳起点 |

### 3.4 预期收益估算

假设 N 型信号策略平均准确率 **55%**（保守估算），盈亏比 **1:1.5**（即平均每手盈利 ¥X，平均每手亏损 ¥1.5X）：

**每笔交易期望损益：**
- 胜率 55%，每笔盈利 = ¥E
- 败率 45%，每笔亏损 = ¥1.5E
- 期望值 = 0.55 × E - 0.45 × 1.5E = 0.55E - 0.675E = **-0.125E**
- 这个期望值是**负的**，意味着仅靠全品种平均准确率无法盈利。

**需要依赖高信度品种筛选：**
- SmartFilter 筛选后，能源化工板块加权 +15%，高信度品种准确率 ≥ 80%
- RB: 100%, Y: 100%, P: 94.2%, EB/L/PG/TA: 84.6%
- **如果只交易高信度品种 (≥80%)，平均准确率 ~85%**：
  - 期望值 = 0.85 × E - 0.15 × 1.5E = 0.85E - 0.225E = **+0.625E**
  - 即每笔平均盈利 E 的 62.5%

**保守收益估算（只交易高信度品种）：**

| 场景 | 品种筛选 | 预期准确率 | 月交易次数 | 单笔盈利E | 月预期净收益 | 月交易成本 | 月净利润 |
|:---:|:-------:|:--------:|:---------:|:--------:|:----------:|:--------:|:------:|
| 保守 | 仅≥80%品种 | 85% | 5 | ¥1,000 | ¥3,125 | ¥250 | **+¥2,875** |
| 中等 | 中等信度+筛选 | 70% | 20 | ¥1,000 | ¥5,000 | ¥900 | **+¥4,100** |
| 积极 | 全品种+SmartFilter | 55% | 40 | ¥1,000 | -¥5,000 | ¥1,800 | **-¥6,800** |

> **关键洞察：** SmartFilter 的价值在于**抑制低信度品种的交易**。全品种无差别交易会导致亏损。只有严格执行品种筛选（只交易历史准确率 ≥ 60% 的品种）才能实现正期望值。

## 4. 实盘路径建议

### 4.1 最小可行资金量

```
┌────────────────────────────────────────────────────┐
│      最小可行资金量：¥50,000 (约 $7,000)           │
├────────────────────────────────────────────────────┤
│  保证金储备: ¥15,000  (1手保证金, 30%仓位)        │
│  安全垫:     ¥30,000  (应对追加保证金和连续亏损)   │
│  运营成本:   ¥5,000   (6个月数据和API费用储备)     │
├────────────────────────────────────────────────────┤
│  预期月收益: ¥700~2,000 (拉面盈利级别)            │
│  预期回撤:   ¥5,000~10,000 (20%最大回撤)           │
└────────────────────────────────────────────────────┘
```

**建议起点：¥50,000 ~ ¥100,000**
- 低于 ¥50,000：保证金不够（仅够1手螺纹钢），无法分散风险
- ¥50,000：可以支撑1-2手高信度品种（如RB、M），月均预期收益 ¥700-2,000
- ¥100,000：可以支撑3-4手，有初步分散能力，年化预期 15-20%
