# Plan: P1-B — N 型结构参数优化 + 基线对照 + Paper Trading 设计 + 信号管道部署

## 目标
在 N 型结构方向预测力已验证（82.6% 1d / 93.2% 5d）的基础上，推进到可交易信号阶段：

1. 验证参数鲁棒性（swing_window_size）
2. 对比简单动量基线（证明 N 型结构有超额价值）
3. 设计 Paper Trading 策略（N 型方向 + 15m 突破入场）
4. 部署自动信号生成管道

## 背景

### S.2.1 突破性发现（已完成 ✅）
- 7 品种、1,316 个结构、10+ 年数据回测
- 1d 方向准确率 82.6% / 5d 93.2% / 10d 76.4%
- Walk-Forward 验证通过（训练/验证差距均值仅 4.1pct）
- P 棕榈油最强（92.5% WFA valid 1d），EB 最弱（70.8%）

### 正式 GO 决策
基于数据的判断：82.6% >> 75% 阈值，WFA 验证无过拟合 → **P1-B 正式 GO**

### 品种优先级
1. **P 棕榈油** — 最强稳定性（92.5% WFA），优先用于参数优化
2. **RB/Y/PG** — 验证稳健性（81-83% WFA），次优先
3. **EB** — 最弱（70.8%），作为下限参考不优先投入

## 拆解步骤

整个 P1-B 拆为 **5 个 Step（Cycle）**：

---

### S.2.2.1: 参数优化实验（swing_window_size）— ≈15min

| 项目 | 内容 |
|------|------|
| **目标** | 确定最佳 swing_window_size |
| **实验范围** | [3, 5, 7, 10, 14, 20] |
| **测试品种** | P（主力）+ RB（验证） |
| **评估指标** | 1d/5d 准确率、结构数量、最大回撤 |
| **产出** | `futures/param_optim.py` 或直接扩展 `n_backtest.py` |

**具体方案：**
- 在 `run_symbol_backtest` 基础上增加 `swing_window` 参数
- 对每个窗口值跑 P 和 RB 的全量回测
- 输出准确率对比表 + 结构数量表
- **关注点**：窗口太大 → 结构太少；窗口太小 → 噪声过多
- **可接受阈值**：最优值相比默认（5）提升 ≥ 2pct 才切换

**依赖**：无需前置

---

### S.2.2.2: 简单动量基线对照 — ≈20min

| 项目 | 内容 |
|------|------|
| **目标** | 证明 N 型结构有超额预测力，非简单趋势跟踪 |
| **基线定义** | 过去 N 日收益率方向（N=5,10,20,60）作为方向预测 |
| **测试品种** | 全 7 品种 |
| **统计方法** | McNemar 检验（配对分类） |
| **产出** | `futures/momentum_baseline.py` + `docs/plan-p1-b.md` 追加结果 |

**具体方案：**
- 用同一 1d Kline 数据集
- 简单动量预测器：`direction = +1 if close[t] > close[t-N] else -1`
- 对比 N 型结构 vs 各窗口动量的准确率
- **最低可接受**：N 型至少比所有动量窗口高出 ≥ 5pct

**依赖**：S.2.2.1 完成（参数优化后 N 型性能更准确）

---

### S.2.2.3: Paper Trading 策略设计（N 型 + 15m 突破入场）— ≈20min

| 项目 | 内容 |
|------|------|
| **目标** | 设计完整交易策略规则，不写代码 |
| **入场逻辑** | 1d N 型确认方向 → 等待 15m Kline 突破 C 点极值 → 入场 |
| **出场逻辑** | A/B/C 点距离的 1:1 或 1:2 目标位，或 ATR 跟踪止损 |
| **止损规则** | 入场价下方 N 型 C 点或 1.5×ATR |
| **仓位规则** | 单笔 ≤ 20% 可用保证金，总敞口 ≤ 50% |
| **产出** | `docs/strategy/p1-b-paper-trading-spec.md`（策略规格文档） |

**关键设计点：**
- 多时间帧衔接：1d N 型确定方向 → 15m Kline 确定精确入场
- 15m 突破 = 价格超过最近 N 型 C 点极值（或 A 点—取决于结构方向）
- 信号有效期：N 型确认后 10 个交易日
- 避免假突破：要求 15m Kline 收盘价确认突破，非盘中瞬间

**依赖**：S.2.2.2 完成（确认 N 型超额价值后才值得设计策略）

### S.2.2.4: Paper Trading 回测框架 — ≈20min

| 项目 | 内容 |
|------|------|
| **目标** | 实现 N 型 + 15m 突破的完整回测 |
| **代码** | `futures/n_paper_backtest.py` |
| **数据** | 1d Kline（N 型检测）+ 15m Kline（突破入场） |
| **评估** | 准确率、盈亏比、最大回撤、Sharpe Ratio |
| **产出** | `futures/n_paper_backtest.py` + 回测结果 |

**具体方案：**
- 扩展 `n_backtest.py` 逻辑
- 在确认 N 型方向后，加载对应 15m Kline 数据窗
- 模拟突破入场 → 跟踪出场
- 记录每笔交易的入场价、出场价、收益、持有时间

**依赖**：S.2.2.3 完成（策略规则定稿）

### S.2.2.2: 简单动量基线对照 — 实验结果

| 项目 | 内容 |
|------|------|
| **完成时间** | 2026-06-14 Cycle #143 |
| **产出的代码** | `futures/momentum_baseline.py` |
| **测试品种** | P, RB, Y, EB, L, PG, TA（7 品种，1,316 个 N 型结构） |
| **动量窗口** | N=5, 10, 20, 60 日 |
| **统计方法** | McNemar 配对检验（连续校正 χ²） |

#### N 型 vs 各动量窗口 — 跨品种聚合

| 动量窗口 | N 型 1d | 动量 1d | 差距 | N 型 5d | 动量 5d | 差距 | McNemar χ² | 显著 |
|:--------:|:-------:|:-------:|:----:|:-------:|:-------:|:----:|:----------:|:----:|
| 5d | 82.6% | 18.8% | **+63.8pct** | 93.2% | 11.1% | **+82.1pct** | 582.7 | ✅ p<0.01 |
| 10d | 82.6% | 38.2% | **+44.4pct** | 93.2% | 38.1% | **+55.1pct** | 422.7 | ✅ p<0.01 |
| 20d | 82.6% | 53.3% | **+29.3pct** | 93.2% | 55.9% | **+37.3pct** | 272.6 | ✅ p<0.01 |
| 60d | 82.7% | 52.1% | **+30.6pct** | 93.4% | 52.8% | **+40.6pct** | 271.7 | ✅ p<0.01 |

#### 关键发现

1. **N 型结构在所有动量窗口上具有压倒性优势**
   - 最小 1d 差距：**+29.3pct** (vs 20d 动量) — 远超 ≥5pct 阈值
   - 最小 5d 差距：**+37.3pct** (vs 20d 动量)
   - 所有 20 个 (4 窗口 × 5 lookahead) McNemar 检验全部 p<0.01

2. **简单动量 ≈ 抛硬币**
   - 5d 动量仅 18.8% 1d 准确率（趋势延续性太短）
   - 10-60d 动量 38-53%（仍远低于 N 型 82.6%）

3. **结论明确**：✅ N 型结构具有超额预测力，非简单趋势跟踪
   - 差距 ≈ 30-80pct，是数量级的差异，非边缘提升
   - 所有 McNemar 检验 p < 0.01，统计显著性无任何争议

#### 对后续步骤的影响
- ✅ 确认 N 型结构值得进一步投入 → **S.2.2.3（策略设计）可以启动**
- ✅ 确认 N 型在 2d-5d 窗口最强（89-93%），策略设计时优先关注
