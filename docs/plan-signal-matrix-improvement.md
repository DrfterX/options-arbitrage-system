# P0 信号矩阵面板改良计划

## 目标
对已有期货/期权两个独立信号面板进行 UX/UI 进阶改良，提升专业交易仪表盘的视觉质量和交互体验。

## 当前状态
- ✅ `futures.drifter.indevs.in` — 期货信号矩阵（N型共振，63品种，全功能：矩阵表/持仓/回测/健康）
- ✅ `options.drifter.indevs.in` — 期权信号仪表盘（IV监控，策略信号，三栏布局）
- ✅ `signals.drifter.indevs.in` — 统一入口页（显示期货汇总数据）
- 三个页面均已稳定运行（200 OK）

## 改良方向（经 product-norman + ui-duarte + interaction-cooper 评估）

### 方向 A：入口页重设计（信号矩阵统一门户）
**问题**：当前 `signals.drifter.indevs.in` 只显示期货数据，没有体现"期货 × 期权"双产品定位。访客可能不知道有独立面板。

**方案**：
- 改为门户/导航页，两边各展示一个快照 widget
- 期货快照：Stats bar + 板块偏度
- 期权快照：IV汇总 + Top 策略
- 清晰的入口按钮到两个独立面板

### 方向 B：跨面板导航系统
**问题**：目前跨面板导航靠一个文本链接按钮，不够醒目，新用户可能找不到。

**方案**：
- 在所有三个页面添加统一的顶部导航栏（Tab 式）
- "📊 总览 | 📈 期货 | 📉 期权" 三级 Tab
- 当前页面高亮，点击直接切换

### 方向 C：数据加载体验优化
**问题**：面板包含大量 JS 异步请求（positions/backtest/health/SmartFilter 等），首次加载白屏时间长。

**方案**：
- 添加 CSS skeleton loading 占位
- 关键数据（矩阵、IV图表）优先加载
- 次要数据（回测、持仓历史）延迟加载
- 添加数据刷新时间戳和加载状态指示器

### 方向 D：视觉统一与专业感提升
**问题**：当前配色在期货（金色#f0b429）和期权（紫色#a855f7）之间不一致。

**方案**：
- 保持品牌差异性但统一底层设计 token（间距、圆角、字体、阴影）
- 期货保持金色暖调，期权保持紫色冷调
- 统一数据卡片、表格、按钮的交互反馈
- 添加数据密度切换（紧凑/舒适模式）

## 设计决策（frontend-design.md 指导）

1. **Tone**: 专业交易仪表盘 — 数据密集、精密、克制。类似 Bloomberg Terminal 的现代版。
2. **Color**: 期货=琥珀金（#f0b429→暖），期权=紫罗兰（#a855f7→冷），统一底层深色主题
3. **Typography**: 数据值使用等宽字体（font-mono），标题保持现有 sans-serif
4. **Motion**: 页面切入动画、数据刷新时的数字过渡动画、hover 状态反馈
5. **Differentiation**: "实时 N 型结构可视化" + "IV 百分位闪电图" — 两个面板各有不同的记忆点
